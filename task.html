Things I Have Done So Far

This is mini model of Swimming Pool Water Monitoring System named as Clean Wave. It consists of a ph sensor , TDS sensor and a rain sensor. the ph sensor regulates the ph of the water by keeping a check on the ph level and alerting the user , if there are any disturbances with the ph of water , by lightening the bulb. secondly TDS sensor is used to keep a check on the appropriate amount of dissolved elements in water. it alarms the user id there is any change by a buzzer sound. and the last but not the least we have a rain sensor ehich stects heavey rain and cover the pool with shed preventing it from acidic rain water.

The displayed model is an Agrobot. This is designed to replace human labour and make work easier.They are to be used to automate manual tasks, such as weed or bracken spraying, where the use of tractors and other human-operated vehicles is too dangerous for the operators. The mechanical design consists of different mechaisms such as seed sowing, irrigation, weed cutter, leveller and a plougher. there is a humidity sensor which will sense the moisture level in soil and automatically irrigate the feilds according to the need. the seed sower will sow seeds leaving eqaul gaps in between.
